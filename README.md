# How to USE:
**To use the GUI:**
- Run main.py to start a Flask server.
- Enter the returned color codes to get the optimal word.

**To auto-solve today's Wordle:**
- Run AutoSolveBot.py.
- This script uses Selenium to automatically solve the Wordle puzzle for the day.
